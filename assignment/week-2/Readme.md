# Week 2 Assignments

This folder contains JavaScript programs that helped me practice array methods, functions, objects, strings, and basic problem solving.

## Assignments List

### 1-map_method.js
Program using the `map()` method to create a new array by modifying existing array elements.

### 2-filter_method.js
Program using the `filter()` method to filter elements based on a condition.

### 3-reduce_method.js
Program using the `reduce()` method to calculate values like sum from an array.

### 4-foreach_method.js
Program using the `forEach()` method to iterate through array elements.

### 5-find_method.js
Program using the `find()` method to search and return an element from an array.

### 6-findindex_method.js
Program using the `findIndex()` method to get the index of an element.

### 7-some_every_methods.js
Program using `some()` and `every()` methods to check conditions in arrays.

### 8-sort_method.js
Program to sort array elements using the `sort()` method.

### 9-string_operations.js
Programs based on string operations and string methods.

### 10-object_operations.js
Programs using objects and accessing/updating object values.

### 11-array_destructuring.js
Program demonstrating array destructuring in JavaScript.

### 12-object_destructuring.js
Program demonstrating object destructuring.

### 13-spread_operator.js
Program using spread operator with arrays and objects.

### 14-rest_operator.js
Program using rest operator in functions and arrays.

### 15-arrow_functions.js
Programs using arrow function syntax in JavaScript.
