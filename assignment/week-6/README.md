# Week 6 Assignments

This week focused on building interactive React applications using forms, state management, props, dynamic rendering, and Context API. The assignments helped me understand how React handles user input, component communication, and global state management.

---

## Assignment-1: User Form & Dynamic Table

In this assignment, I created a React application where users can enter details through a form and display the submitted data dynamically in a table.

### Concepts Practiced
- React components
- useState hook
- Handling form inputs
- Event handling
- Dynamic table rendering
- Updating UI based on user input

### Features
- Add user details using a form
- Store data in state
- Dynamically render table rows
- Simple responsive frontend structure

---

## Assignment-2: Employee Management System

This assignment was a full-stack Employee Management System project with frontend and backend integration.

### Concepts Practiced
- React frontend
- Node.js and Express backend
- API integration
- CRUD operations
- State management
- Component-based architecture

### Features
- Add employee details
- Update employee records
- Delete employee data
- Display employee information dynamically
- Frontend and backend communication

This assignment helped me understand how frontend and backend applications work together in a real-world project structure.

---

## Assignment-3: Global State with Context API

This assignment focused on React Context API and global state management.

### Concepts Practiced
- Context API
- Global state sharing
- Component communication
- useContext hook
- Avoiding prop drilling

### Features
- Shared global state between components
- Cleaner component structure
- Easier data access across the application

This assignment made it easier to understand how larger React applications manage shared data efficiently.
