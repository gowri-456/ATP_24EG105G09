# Week 1 Assignments - JavaScript Basics

This folder contains JavaScript practice programs for basic programming logic, arrays, functions, and objects.

## Files

| File | Description |
| --- | --- |
| `q-1.js` | Finds the larger number between two given numbers. |
| `q-2.js` | Finds the largest number among three given numbers. |
| `q-3.js` | Calculates the sum of marks in the array `[90, 78, 65, 98]`. |
| `q-4.js` | Finds the smallest element in a marks array. |
| `q-5.js` | Defines a function that receives three numbers and returns the largest number. |
| `q-6.js` | Defines a function that receives an array and calculates the sum of its elements. |
| `q-7.js` | Searches for an element in an array and returns its index, or `-1` when it is not found. |
| `q-8.js` | Performs operations on an employee array using objects and arrays. |

## Employee Array Operations

The `q-8.js` file works with an `employees` array and performs these operations:

- Inserts a new employee named `Teja` at the second position.
- Deletes the employee named `Kiran`.
- Changes Sneha's last mark from `95` to `75`.

## Concepts Practiced

- Variables using `let` and `const`
- Conditional statements using `if`, `else if`, and `else`
- Loops using `for`
- Arrays and array indexing
- Functions with parameters and return values
- Objects inside arrays
- Array operations using `splice`

## How to Run

Run any file with Node.js:

```bash
node q-1.js
```

Replace `q-1.js` with the file you want to run.
