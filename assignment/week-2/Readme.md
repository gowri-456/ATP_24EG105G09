# Week 2 Assignments

This folder contains JavaScript programs focusing on array methods, the spread operator, asynchronous JavaScript, and Object-Oriented Programming.

## Assignments List

### [q-1.js](./q-1.js)
**Shopping Cart Summary:** Uses array methods (`filter()`, `map()`, `reduce()`, `find()`, and `findIndex()`) to process a shopping cart for an e-commerce website.

### [q-2.js](./q-2.js)
**Student Performance Dashboard:** Processes student marks and generates grades for a college result analysis system using array methods.

### [q-3.js](./q-3.js)
**Employee Payroll Processor:** Builds a salary processing module in a company HR app to calculate net salaries, total payout, and find specific employees.

### [q-4.js](./q-4.js)
**Movie Streaming Platform:** Implements a movie recommendation system to filter genres, format ratings, calculate average ratings, and search movies.

### [q-5.js](./q-5.js)
**Bank Transaction Analyzer:** Builds a bank statement summary by analyzing credit and debit transactions to calculate the final balance.

### [q-6.js](./q-6.js)
**Spread Operator Practice:** Demonstrates copying and merging arrays, and updating object properties using the spread operator (`...`).

### [q-7.js](./q-7.js)
**Asynchronous JavaScript:** Practices `setTimeout` and `setInterval` by simulating exam submission evaluation and OTP resend timers.

### [q-8.js](./q-8.js)
**Library Book Management System:** Creates a `Book` class to manage a collection of books with properties and methods to borrow, return, and get book information.

---

## Concepts Learned

- Array methods (`map()`, `filter()`, `reduce()`, `find()`, `findIndex()`)
- Spread operator for Arrays and Objects
- Asynchronous JavaScript (`setTimeout`, `setInterval`)
- Object-Oriented Programming (Classes, Constructors, Methods, Properties)

---

## How to Run

Run the files using Node.js in your terminal:

```bash
node q-1.js
```
