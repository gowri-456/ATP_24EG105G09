# Week 3 Assignments - REST API and Models

This folder contains backend assignments using Node.js, Express.js, MongoDB, and Mongoose.

## Files

### ProductModel.js
Created a product schema and model using Mongoose.

Fields used:
- productId
- productName
- price
- brand

Validations were also added for required fields and price range.

### ProductAPI.js
Implemented REST API operations for products.

Operations:
- Create product
- Read all products
- Read product by productId
- Update product by productId
- Delete product by productId

## Concepts Learned

- REST APIs
- MongoDB
- Mongoose Schema & Model
- CRUD operations
- Validations
- Express routing

## How to Run

Install dependencies:

npm install


Start server:


node server.js
